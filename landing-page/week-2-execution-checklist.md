## Outreach Message Quick Reference

### Template Selection

| Prospect Type | Template |
|--------------|----------|
| Former colleague | Template 1 (温暖接触) |
| WeChat group member | Template 2 (群首次接触) |
| Asked AI question | Template 3 (针对性分享) |
| Article reader follow-up | Template 4 (跟进消息) |
| Asking for referrals | Template 5 (请求转介绍) |

### Key Principles

1. **Value first** - Share article/insight before asking
2. **Personalize** - Reference specific context
3. **Clear CTA** - One ask per message
4. **Respectful follow-up** - 2-3 days, max 3 times
5. **Track everything** - CRM updated same day

## Success Triggers & Pivots

### Celebrate When 🎉
- First 100 landing page visitors
- First 10 leads in CRM
- First discovery call booked
- First article hits 500+ reads
- First positive response from outreach

### Pivot When ⚠️
- 5 days with zero outreach responses → Revise messaging
- 10 messages with zero replies → Change value proposition
- Article gets no engagement after 3 days → New headline/format
- Calls booked but no proposals → Review qualification criteria
