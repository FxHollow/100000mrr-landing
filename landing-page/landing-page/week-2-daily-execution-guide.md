# Week 2 Daily Execution Guide - CMO

## 第二周每日执行指南

**周期**: Apr 1-7, 2026
**目标**: 20 次外联，5 次回复，2-3 个电话，1 个 proposal

---

## Day 1 (周一，Apr 1) - Launch Day

### 上午 (9:00-12:00)
- [ ] **部署 landing page**
  - GitHub Pages: 创建 repo → push → 启用 Pages
  - 或 Vercel: `vercel --prod` (需 login)
  - 测试链接：手机 + 桌面都检查
- [ ] **设置 analytics**
  - Google Analytics 或 百度统计
  - 在 index.html 中添加 tracking code
- [ ] **发布文章 1 到知乎**
  - 使用 `zhihu-article-1.md` 版本
  - 标题：5 个 AI 集成实战案例，帮中小企业节省 30-50% 成本
  - 添加 5-8 个话题标签
  - 发布后 2 小时内回复评论

### 下午 (2:00-6:00)
- [ ] **发布文章 1 到掘金**
  - 使用 `juejin-article-1.md` 版本
  - 标题：5 个 AI 集成实战案例，中小企业如何用 AI 降本 30-50%？
  - 添加技术标签
- [ ] **加入 2-3 个微信群**
  - 搜索关键词：电商老板群、AI 应用交流群、创业者社群
  - 观察群动态，不要急着发广告
  - 记录群名称和活跃度

### 晚上 (8:00-9:00)
- [ ] **检查当日数据**
  - 知乎：阅读量、点赞、评论
  - 掘金：阅读量、收藏、评论
  - Landing page：访客数（如有 analytics）
- [ ] **回复所有评论和私信**
  - 2 小时内回复是目标
  - 引导有兴趣的私聊

**Day 1 成功标准**:
- Landing page 上线 ✅
- 知乎 + 掘金文章发布 ✅
- 2-3 个微信群加入 ✅
- 收到第一波读者反馈 ✅

---

## Day 2 (周二，Apr 2) - Community Building

### 上午 (9:00-11:00)
- [ ] **发布小红书笔记 #1**
  - 主题：AI 案例 1（客服自动化）
  - 封面：Canva 制作，竖版 3:4
  - 文案：500 字内，5-8 个标签
  - 引导私信咨询
- [ ] **回复知乎/掘金评论**
  - 逐一回复前天的评论
  - 对高价值评论深入交流

### 下午 (2:00-5:00)
- [ ] **加入更多微信群（目标 5 个）**
  - 再找 2-3 个群
  - 观察群规和活跃度
  - 记录群主/管理员微信
- [ ] **准备 outreach 名单**
  - 从 CEO/CTO 获取 warm intro 名单
  - 整理前 10 个目标客户
  - 在 CRM 中建立记录

### 晚上 (8:00-9:00)
- [ ] **在 1-2 个群里分享价值**
  - 回答 1-2 个 AI 相关问题
  - 不hard sell，只分享见解
  - 有人问再发文章链接

**Day 2 成功标准**:
- 小红书笔记 #1 发布 ✅
- 5 个微信群加入 ✅
- 10 个 prospects 名单 ready ✅
- 群里开始建立存在感 ✅
