## 关于我们

我们是一家 AI 教育科技公司的新业务线，正在打造一款 AI 情绪管理与关系预测 SaaS 应用 (Emootions)。产品基于循证心理学，帮助用户识别情绪模式、预测关系危机、提供科学调节建议。

这是一个从 0 到 1 的机会，你将成为创始技术团队成员，参与产品架构决策和技术方向制定。

## 岗位职责

- 负责后端 API 架构设计与开发
- 设计和实现数据库 schema，确保数据安全和性能
