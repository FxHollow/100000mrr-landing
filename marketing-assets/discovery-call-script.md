# Discovery Call Script - 30 分钟电话咨询

## Discovery Call 完整流程

---

## 会前准备 (5 分钟)

### 了解客户背景
- [ ] 查看公司官网/LinkedIn
- [ ] 了解行业和规模
- [ ] 查看之前的沟通记录
- [ ] 准备 1-2 个类似案例

### 技术检查
- [ ] 测试会议链接/电话
- [ ] 打开 ROI calculator
- [ ] 准备好 case studies
- [ ] 打开 CRM 准备记录

---

## Call 流程 (30 分钟)

### 开场白 (2 分钟)

**话术**:
```
[客户名字] 总好，感谢您抽出时间！

我是 [名字]，100000MRR 的 CMO。在开始之前，我想简单介绍一下今天的议程：

1. 先了解贵公司当前的运营情况和面临的挑战（10 分钟）
2. 看看 AI 集成如何帮助解决这些问题（10 分钟）
3. 讨论下一步建议，无论是否合作都会有具体建议（10 分钟）

整个咨询是免费的，不会有 hard sell。我们希望通过交流，帮您判断 AI 集成是否值得投入。

您看这样可以吗？
```

**目标**: 建立信任，设定预期

---

### 第一部分：了解现状 (10 分钟)

#### 核心问题

**Q1: 业务概况**
```
可以先简单介绍一下贵公司的业务吗？
- 主要产品/服务是什么？
- 团队规模大概多少人？
- 目前在 [行业] 做了多久？
```

**Q2: 当前痛点**
```
您目前运营中最大的挑战是什么？
- 哪些流程最耗时/耗力？
- 客户投诉最多的是什么？
- 如果有一个魔法可以解决一个问题，您希望是什么？
```

**Q3: 现有工具/流程**
```
目前用什么工具/系统来管理这些流程？
- CRM/ERP 系统？
- 客服工具？
- 内部协作工具？
```

**Q4: AI 经验**
```
之前有尝试过 AI 相关的工具或方案吗？
- 用过什么？
- 效果如何？
- 有什么顾虑？
```

**Q5: 预算和决策**
```
如果方案可行，您心里的预算范围大概是多少？
决策流程是怎样的？需要和谁讨论？
```

#### 倾听要点

记录以下信息：
- **痛点优先级**: 哪个最痛？
- **决策链**: 谁是最终决策者？
- **预算范围**: 是否有清晰预期？
- **时间紧迫度**: 为什么是现在？

---

### 第二部分：方案介绍 (10 分钟)

#### 连接痛点与方案

**话术模板**:
```
根据刚才的交流，我理解您主要面临 [痛点 1]、[痛点 2] 这些挑战。

我们帮助过一家 [类似公司] 解决了类似问题，他们通过 [AI 方案] 实现了：
- [量化成果 1]
- [量化成果 2]
- [量化成果 3]

具体到您的情况，我认为可以从 [具体场景] 入手，因为：
1. [理由 1 - 高 ROI]
2. [理由 2 - 低风险]
3. [理由 3 - 快速见效]
```

#### 展示案例

**准备 2-3 个相关案例**:

| 行业 | 痛点 | 方案 | 成果 |
|------|------|------|------|
| 电商 | 客服人力成本高 | AI 客服自动化 | 80% 自动响应，节省 3 FTE |
| 制造 | 合同审核慢 | AI 合同审核 | 从 3 天→4 小时 |
| 分销 | 销售文档耗时 | 文档自动生成 | 节省 90% 时间 |

#### ROI 演示

使用 ROI calculator 展示:
```
当前成本:
- 人力：X 人 × ¥Y/月 = ¥Z/月
- 时间：X 小时/周 × ¥Y/小时 = ¥Z/月

AI 方案成本:
- 开发：一次性 ¥X
- API: ¥Y/月

回本周期: Z 个月
年度节省：¥X
```

---

### 第三部分：下一步 (8 分钟)

#### 评估 fit

**合格信号** (至少满足 3 个):
- [ ] 痛点清晰且量化
- [ ] 有决策权/预算
- [ ] 时间紧迫 (3 个月内)
- [ ] 认可 ROI
- [ ] 愿意提供数据/配合

**不合格信号**:
- [ ] 只是"了解一下"
- [ ] 无明确预算
- [ ] 决策链复杂 (>6 个月)
- [ ] 痛点不清晰

#### 提议下一步

**如果 fit 好**:
```
基于今天的交流，我认为 AI 集成对贵公司是值得投入的。

我建议的下一步是：
1. 我们安排一次详细的技术评估 (1-2 小时)
2. 我们会提供一份详细的方案书，包含：
   - 具体实施范围
   - 时间周期 (一般 4 周)
   - 投资预算
   - 预期 ROI
3. 如果方案合适，我们可以安排启动会议

您看这周 [时间 1] 或 [时间 2] 方便做技术评估吗？
```

**如果 fit 一般**:
```
感谢今天的交流。基于我们聊的内容，我建议：

1. 您可以先试试 [低成本方案/自助工具]
2. 我们有一篇详细的方法论文章，我发给您参考
3. 如果后续有更复杂的需求，欢迎随时联系

这是文章链接：[链接]
```

**如果不 fit**:
```
感谢您抽出时间。老实说，基于我们今天聊的，我觉得 AI 集成可能不是您当前的优先级。

我建议您可以先 [替代方案]。如果后续情况变化，欢迎再联系。

这是我们的案例合集，供您未来参考：[链接]
```

---

### 收尾 (2 分钟)

**确认下一步**:
```
那我们确认一下下一步：
- [具体行动]
- [时间节点]
- [负责人]

我会在 [时间] 前发送 [会议邀请/方案书/资料]。

还有什么问题吗？
```

**感谢**:
```
再次感谢您的时间！期待 [下一步]。

如有任何问题，随时微信/邮件联系。

祝好！
```

## 常见异议处理

### "我们需要再考虑一下"

**回应**:
```
当然，这是重要的决定。请问主要是哪方面需要再考虑？
- 是方案的技术可行性？
- 是预算方面？
- 还是优先级的问题？

我很乐意针对您的顾虑做进一步说明。
```

### "价格有点高"

**回应**:
```
理解预算的考虑。可以问一下，您预期的预算范围是多少吗？

另外，从 ROI 角度看，这个方案预计 [X 个月] 回本，年度节省 [¥Y]。
如果预算有限，我们可以考虑分阶段实施，先做 [高 ROI 的部分]。

您觉得这样可以吗？
```

### "我们之前试过 AI，效果不好"

**回应**:
```
理解您的顾虑。可以具体说说之前尝试的是什么吗？哪里效果不理想？

AI 在过去 12 个月进步非常大，很多之前做不到的事现在可以了。
也许我们可以从一个小场景开始，快速验证效果，您看如何？
```

### "我们需要和团队讨论"

**回应**:
```
当然，这是团队决策。请问需要和哪些人讨论？

我可以准备一份简版的方案摘要，方便您内部沟通。
或者安排一次和技术/业务团队的答疑会议也可以。

您觉得哪种方式更有帮助？
```

## 成功指标

一次成功的 discovery call 应该有：
- [ ] 客户痛点清晰
- [ ] 客户认可 ROI
- [ ] 明确了决策流程
- [ ] 安排了具体下一步
- [ ] 双方时间投入值得
