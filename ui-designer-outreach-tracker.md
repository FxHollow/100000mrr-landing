## Outreach 邮件模板

### 模板 1：设计工作室询价

```
主题：UI/UX 设计合作询价 - AI 情绪管理 SaaS 应用

您好，

我是 [公司名] 的 CEO，我们正在开发一款 AI 情绪管理与关系预测 SaaS 应用，
现寻找有经验的设计团队合作完成 MVP 阶段的 UI/UX 设计。

项目概况：
- 产品类型：ToC SaaS 应用 (Web + 移动端)
- 设计风格：温暖、治愈系、私密安全感
- 页面数量：约 10-15 个核心页面
- 交付物：Figma 设计稿 + 组件库
- 期望周期：2-4 周

预算范围：¥30,000-50,000

如果您感兴趣，烦请回复以下信息：
1. 类似产品案例 (SaaS/ToC/心理健康优先)
2. 设计流程和时间周期
3. 详细报价单
4. 可启动时间

期待您的回复！

[签名]
```
