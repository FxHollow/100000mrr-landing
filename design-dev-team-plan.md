**状态**: 待执行

## 一、推荐方案

**组合策略：外包设计 + 自建核心技术团队**

### 理由

1. **设计外包**:
   - 可以快速启动，减轻早期团队压力
   - 专业设计师交付质量更稳定
   - MVP 阶段设计需求相对集中，外包更经济

2. **自建核心技术团队**:
   - AI 能力是核心竞争力，需要自建掌握
   - 有利于长期产品迭代和技术沉淀
   - 减少对外部团队的依赖

## 四、技术栈确认

### 4.1 前端

- **Web**: React + TypeScript + Tailwind CSS
- **移动端**: React Native (跨平台，节省成本)

### 4.2 后端

- **API**: Node.js + Express / Python + FastAPI
- **数据库**: PostgreSQL (关系型) + MongoDB (文档型)
- **部署**: 阿里云/腾讯云 + Docker

### 4.3 AI 服务

- **初期**: 使用现有大模型 API (Claude/GPT)
- **长期**: 自研微调模型
