## 笔记 8: AI 提示词模板，直接套用

**封面标题**:
```
📝 AI 提示词模板
这 10 个直接套用
效率翻倍
```

**正文内容**:

整理了 10 个万能 Prompt 模板，直接复制就能用👇

**1. 周报生成**
```
请帮我写一份工作周报，包含：
1. 本周完成的主要任务（3-5 项）
2. 遇到的问题和解决方案
3. 下周工作计划
我的工作：[填写]
```

**2. 邮件写作**
```
请帮我写一封 [类型] 邮件，收件人是 [身份]
主要内容是：[填写]
语气要求：[正式/友好/紧急]
```

**3. PPT 大纲**
```
我要做一个关于 [主题] 的 PPT
受众是：[领导/客户/同事]
请设计 10 页 PPT 的大纲
每页包含标题和 3-5 个要点
```

**4. 文章写作**
```
请帮我写一篇关于 [主题] 的文章
字数：[1000 字]
风格：[专业/轻松/故事化]
发布平台：[公众号/知乎/小红书]
```

**5. 数据分析**
```
我有以下数据：[粘贴数据]
请帮我分析：
1. 主要趋势
2. 异常值
3. 建议行动
```

**6. 会议纪要**
```
以下是会议讨论的要点：[粘贴内容]
请帮我整理成会议纪要，包含：
1. 参会人员
2. 讨论事项
3. 决策结果
4. 待办事项
```

**7. 方案策划**
```
我需要策划一个 [活动/项目]
目标：[填写]
预算：[填写]
时间：[填写]
请帮我设计完整方案
```

**8. 学习总结**
```
我学习了以下内容：[粘贴笔记]
请帮我总结：
1. 核心知识点（3-5 个）
2. 实践建议
3. 延伸阅读
```

**9. 问题拆解**
```
我面临的问题是：[描述问题]
请帮我拆解：
1. 问题的根本原因
2. 可能的解决方案
3. 每种方案的利弊
```

**10. 创意生成**
```
我需要 [数量] 个关于 [主题] 的创意
场景：[填写]
目标受众：[填写]
要求：[创新/实用/可执行]
```

**使用技巧**:
- 越具体越好，给 AI 更多上下文
- 可以要求 AI 输出特定格式
- 不满意可以让 AI 重新生成
