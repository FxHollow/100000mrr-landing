## 二、候选人筛选标准

### 硬性条件

- [ ] 心理学硕士及以上学历
- [ ] 国家二级/三级心理咨询师资格证或注册心理师
- [ ] 3 年以上临床经验
- [ ] 中文母语

### 加分项

- [ ] 伴侣/家庭治疗专长
- [ ] 数字化产品经验（App、数字疗法）
- [ ] AI/科技行业理解
- [ ] 英文文献阅读能力

### 红旗信号（排除）

- [ ] 仅有培训证书无学历背景
- [ ] 过度承诺效果（"保证改善关系"等）
- [ ] 非循证方法取向（占星、塔罗等）
- [ ] 时间投入不足（每周<3 小时）

## 五、面试流程

### 第一轮：初步沟通（30 分钟）

**目的**: 双向了解，确认基本匹配度

**问题清单**:
1. 请介绍一下您的专业背景和主要咨询方向
2. 您是否有过数字化产品/App 相关经验
3. 您对 AI 在心理健康领域的看法
4. 每周可投入的时间
5. 期望的合作方式和报酬

**评估要点**:
- 沟通顺畅度
- 对数字化的开放度
- 时间投入意愿
- 专业度初步判断

### 第二轮：专业评估（60 分钟）

**目的**: 深度评估专业能力

**方式**: 案例讨论 + 产品内容评审

**案例**: 提供一个虚拟用户场景，请候选人给出干预建议

**评估要点**:
- 循证方法运用
- 边界意识（产品 vs 咨询）
- 危机干预思维
